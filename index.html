Help – Set up your Rowe Casa Organics email signature

▶ Watch the How-To Video

Important: Do this once for every email app and computer / phone you use. If you use Gmail + Outlook + a mobile app, you should check and test each one (on each device).
  1. Delete your old signature(s) everywhere you use email (computer + phone).
  2. Fill in Full Name, Job Title, and Work Email (must end with @rowecasaorganics.com).
  3. Click Generate.
  4. Click Copy Signature and paste into your email signature settings.
  5. Send yourself a quick test email to confirm it looks right.

Tip: If you use multiple apps (Gmail + Outlook + mobile), repeat this in each app on each device.

Delete your old signature(s) first so the new one doesn’t stack with the old one.

  1. Gmail (Web): Settings ⚙️ → See all settings → Signature → remove old signature(s) → Save Changes.
  2. Outlook (Desktop): Settings/Preferences → Signatures → remove old signature(s).
  3. Mobile apps: App Settings → Signature → remove old signature.

Use the “Email Signature (Recommended)” section on this page.

  1. Click Generate.
  2. Click Copy Signature.
  3. Paste into your email app’s signature settings.

Admin options are for special use-cases only (last resort). Most users should not use them.

Be sure you are editing the correct Rowe Casa Organics account in each app.

  1. Gmail (Web): Set signature defaults for New emails and Replies/Forwards. Save Changes.
  2. Gmail Mobile App: Select your Rowe Casa Organics account → turn Mobile Signature OFF (so it uses your web signature).
  3. Outlook (Mac): Set default signature for New messages and Replies/Forwards.
  4. Outlook (Windows): File → Options → Mail → Signatures → assign for New messages and Replies/Forwards.
